# دليل نشر نظام إدارة الموظفين (Employee HR System)
# النشر باستخدام Docker و Cloudflare Tunnel على Windows 10

هذا الدليل يحتوي على الخطوات التفصيلية لنقل وتشغيل البرنامج على سيرفر محلي باستخدام Windows 10.

## المرحلة الأولى: التحضير (Preparation)
1. تأكد من تثبيت البرامج التالية على جهاز Windows 10:
   - Docker Desktop (مع تفعيل WSL 2).
   - Git (اختياري لنقل الملفات).
2. قم بنسخ مجلد المشروع `local_server_files` إلى جهازك وفك ضغطه في مسار بسيط (مثل `C:\apps\hr-system`).

## المرحلة الثانية: إعداد قاعدة البيانات والملفات (Environment)
1. افتح ملف `docker-compose.yml` وتأكد من الإعدادات:
   - `SESSION_SECRET`: تم تعيينه لقيمة آمنة.
   - `DATABASE_URL`: مربوط بخدمة `db` الداخلية في دوكر.
2. البرنامج مهيأ لإنشاء الجداول تلقائياً عند التشغيل الأول.

## المرحلة الثالثة: تشغيل البرنامج باستخدام Docker
1. افتح واجهة الأوامر (PowerShell أو CMD) في مجلد المشروع.
2. نفذ الأمر التالي لبناء وتشغيل الحاويات:
   ```bash
   docker-compose up -d --build
   ```
3. انتظر حتى يتم تحميل الصور وبناء التطبيق. يمكنك التأكد من عمل البرنامج بزيارة `http://localhost:5000`.
4. الحساب الافتراضي للمدير:
   - المستخدم: admin
   - كلمة المرور: 123456

## المرحلة الرابعة: ربط الدومين (SHEO) عبر Cloudflare Tunnel
1. قم بتثبيت `cloudflared` على ويندوز (أو استخدمه كحاوية دوكر إضافية).
2. تسجيل الدخول وإنشاء التانال:
   ```bash
   cloudflared tunnel login
   cloudflared tunnel create sheo-tunnel
   ```
3. سيظهر لك `Tunnel ID` (معرف فريد)، قم بنسخه ووضعه في ملف `cloudflared-config.yml` بدلاً من `<YOUR_TUNNEL_ID>`.
4. قم بإنشاء السجل في كلاود فلير لربط الدومين الفرعي SHEO:
   ```bash
   cloudflared tunnel route dns sheo-tunnel sheo.yourdomain.com
   ```
5. تشغيل التانال:
   ```bash
   cloudflared tunnel --config cloudflared-config.yml run
   ```

## ملاحظات هامة:
- تم إصلاح ثغرة تسريب الملفات (Storage Leak) بإضافة معالجة أخطاء شاملة.
- تم إصلاح مشكلة حذف المستخدمين بربط سجلات التدقيق (Audit Logs) بالحذف المتسلسل (Cascade).
- تم تفعيل التحقق الصارم من التواريخ (Date Validation) لمنع إدخال بيانات خاطئة.
- المجلد `storage` في جهازك سيحتوي على كافة ملفات الموظفين المرفوعة، حافظ عليه كنسخة احتياطية.
