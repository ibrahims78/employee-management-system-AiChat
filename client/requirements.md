## Packages
recharts | Dashboard charts and data visualization
xlsx | Exporting employee data to Excel
docx | Generating Word documents for employee profiles
file-saver | Saving generated files to the client
date-fns | Date formatting and manipulation
lucide-react | Icons for the UI (already in base, but listing for clarity)
clsx | Utility for conditional classes
tailwind-merge | Utility for merging tailwind classes

## Notes
The application uses a custom authentication system (not Replit Auth).
API endpoints are defined in server/routes.ts and shared/routes.ts.
RTL support is mandatory for all pages.
