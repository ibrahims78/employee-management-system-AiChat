ProTeach ليس مجرد تطبيق — إنه منظومة متكاملة تعمل على خادم سحابي مخصص في Google Cloud، تضم ثلاثة أنظمة فرعية تعمل معاً بشكل متزامن وآمن.

👔

نظام إدارة الموظفين (HR)

إدارة شاملة لبيانات الموظفين، الوثائق، التقارير، والسجلات الوظيفية — كل شيء في مكان واحد قابل للوصول من أي جهاز.

💬

نظام الواتساب الذكي

إرسال واستقبال رسائل الواتساب تلقائياً عبر خطوط متعددة — نصوص، صور، ملفات، وصوتيات — مع لوحة تحكم كاملة.

⚡

محرك الأتمتة (n8n)

أتمتة أي عملية: إرسال تلقائي للموظفين، تنبيهات، تقارير دورية، وتكامل مع أي نظام خارجي — بدون أي كود.

الأنظمة الثلاثة

تفاصيل كل نظام ومميزاته

كل نظام مستقل بقاعدة بياناته الخاصة، لكنها مترابطة عبر شبكة داخلية آمنة.

👔

نظام إدارة الموظفين

HR Employee Management System

  • إضافة وتعديل وحذف سجلات الموظفين بسهولة
  • رفع وتخزين وثائق الموظفين (عقود، شهادات، هوية)
  • تصدير تقارير Excel وWord بنقرة واحدة
  • بحث وفلترة متقدمة حسب القسم والمسمى والحالة
  • لوحة تحكم مع إحصائيات ومؤشرات أداء
  • واجهة عربية كاملة وسريعة الاستجابة
  • نسخ احتياطي يومي تلقائي لقاعدة البيانات
  • وصول آمن عبر HTTPS من أي مكان
💬

نظام إدارة الواتساب

WhatsApp Multi-Session Manager

  • تشغيل أكثر من خط واتساب في نفس الوقت
  • إرسال نصوص، صور، فيديو، صوت، وملفات PDF
  • API كاملة للتكامل مع أي نظام أو ورك فلو
  • سجل كامل لجميع الرسائل المرسلة والمستقبلة
  • دعم أرقام دولية بصيغة E.164
  • آلية retry تلقائية عند فشل الإرسال
  • لوحة تحكم للمشرفين لمراقبة الجلسات
  • مفاتيح API للتحكم بالصلاحيات
⚡

محرك الأتمتة n8n

Workflow Automation Engine

  • أتمتة بصرية بدون كود — اسحب وأفلت
  • جدولة إرسال رسائل واتساب للموظفين تلقائياً
  • تنبيهات تلقائية عند انتهاء العقود أو الإجازات
  • تكامل مع Google Sheets, Email, Telegram وغيرها
  • تشغيل عبر Webhook أو جدول زمني أو حدث
  • أكثر من 400 خدمة وتطبيق جاهز للتكامل
  • واجهة بصرية لبناء ورك فلو معقدة ببساطة
  • سجل تشغيل لكل ورك فلو للمراجعة والتتبع
🌐

البنية التحتية الأمنية

Nginx + Cloudflare + SSL + Portainer

  • حماية Cloudflare كاملة ضد DDoS والهجمات
  • شهادات SSL تتجدد تلقائياً (Let's Encrypt)
  • Nginx كـ reverse proxy يوجه الطلبات بأمان
  • كل خدمة على نطاق فرعي مستقل (hr / n8n / wa)
  • Portainer لمراقبة وإدارة الحاويات بصرياً
  • PostgreSQL لكل نظام — عزل تام للبيانات
  • جدار حماية UFW — فقط المنافذ 80/443/22 مفتوحة
  • نسخ احتياطي يومي تلقائي عند 3:00 AM

مجالات الاستخدام والاستفادة

المنظومة مصممة لأي مؤسسة أو شركة تحتاج إدارة موظفين مع تواصل فعّال وأتمتة للعمليات.

1

🏫 المدارس والمراكز التعليمية

إدارة ملفات المعلمين والموظفين الإداريين، إرسال الجداول والإشعارات للمعلمين عبر الواتساب تلقائياً، وأتمتة التقارير الأسبوعية والشهرية.

2

🏥 العيادات والمستشفيات الصغيرة

إدارة ملفات الكوادر الطبية، إرسال تذكير المرضى بالمواعيد عبر الواتساب، وتنبيه الفريق بالمستجدات والتعليمات.

3

🏢 الشركات والمكاتب التجارية

ملفات الموظفين والعقود والتأمينات، إرسال كشوف الرواتب والإشعارات عبر الواتساب، وأتمتة تجديد العقود والإجازات.

4

🏗️ شركات المقاولات والإنشاءات

إدارة عمال المواقع المتعددة، إرسال التعليمات اليومية وجداول العمل عبر الواتساب، وتتبع بيانات الدوام والإجازات.

5

🛒 سلاسل المتاجر والتجزئة

إدارة موظفي الفروع المتعددة من مكان واحد، إرسال إشعارات المبيعات والمخزون للمدراء، وتقارير الأداء الأوتوماتيكية.

6

🏛️ الجهات والمنظمات غير الربحية

إدارة المتطوعين والكوادر، التواصل الجماعي عبر الواتساب للأعضاء، وأتمتة التقارير الدورية وتجميع البيانات.

الفوائد المرجوّة

ماذا تكسب من هذه المنظومة؟

فوائد ملموسة تنعكس مباشرة على الإنتاجية وتوفير الوقت والتكلفة.

⏱️

توفير الوقت

ما كان يستغرق ساعات من العمل اليدوي (إرسال الرسائل، طباعة التقارير، البحث في الملفات) يصبح تلقائياً بنقرة واحدة.

💰

تخفيض التكاليف

لا حاجة لاشتراكات متعددة في تطبيقات مختلفة — كل شيء في منظومة واحدة على خادم مخصص بتكلفة شهرية ثابتة.

📊

قرارات أفضل

تقارير فورية وإحصائيات دقيقة عن الموظفين والعمليات تساعد الإدارة على اتخاذ قرارات مبنية على بيانات حقيقية.

🔒

أمان البيانات

بياناتك على خادم خاص بك — لا تشارك أحداً بيانات موظفيك. حماية متعددة الطبقات بـ SSL وCloudflare وجدار حماية.

📱

تواصل فعّال

الواتساب هو قناة التواصل الأكثر استخداماً — وصول فوري لجميع الموظفين بدون الحاجة لتطبيق إضافي.

📈

قابلية التوسع

يمكن إضافة موظفين وخطوط واتساب وأتمتة جديدة بسهولة دون تغيير البنية التحتية أو زيادة التكاليف.
